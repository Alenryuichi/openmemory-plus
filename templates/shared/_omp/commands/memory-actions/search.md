# 搜索记忆

在项目级和用户级记忆中进行语义搜索。

## 执行步骤

1. 询问搜索关键词（如果用户未提供）
2. 调用 `search_memory_openmemory` 搜索用户记忆
3. 同时搜索 `.memory/` 目录下的 YAML 文件
4. 按相关度排序，合并展示结果

## 输出格式

```
🔍 搜索结果: "{query}"

📁 项目级结果:
1. [decisions.yaml] 技术决策: 使用 TypeScript
   匹配度: 0.85

👤 用户级结果:
1. 用户偏好使用 pnpm 包管理器
   创建时间: 30天前 | 衰减状态: Active

共找到 2 条相关记忆
```

## 后续操作

用户可以继续操作：

- "删除第 N 条" → 执行删除
- "查看详情" → 展示完整内容
- "更新第 N 条" → 修改记忆内容

## 返回菜单

完成后提示: "还需要其他操作吗？输入 `/memory` 返回菜单"
