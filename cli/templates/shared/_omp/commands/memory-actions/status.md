# 记忆状态

查看项目级和用户级记忆的详细状态。

## 执行步骤

1. 读取 `.memory/` 目录，列出文件和最后修改时间
2. 调用 `list_memories_openmemory` 获取用户记忆
3. 分析衰减状态分布
4. 统计并展示

## 输出格式

```
📊 记忆系统详细状态

📁 项目级 (.memory/)
├── project.yaml    (2024-01-15 更新)
├── decisions.yaml  (2024-01-10 更新)
└── changelog.yaml  (2024-01-14 更新)

👤 用户级 (openmemory)
├── 总记忆数: 15 条
├── 最近添加: "用户偏好中文对话" (1天前)
└── 衰减状态:
    ├── 🟢 Active:  12 条
    ├── 🟡 Aging:   2 条
    └── 🔴 Stale:   1 条

✅ 系统状态正常
```
